Lychee-like fruits

I once claimed that lychee, rambutan, and mamoncillo were basically the same fruit. This page is a more careful correction.

Taxonomic context

Many fruits commonly described as “lychee-like” belong to the family Sapindaceae, within the order Sapindales. However, some fruits outside this family (e.g. mangosteen or certain Annona species) share similar culinary or morphological traits despite being evolutionarily distant.

The lowest taxonomic rank shared by all species on this page is the class Magnoliopsida (dicotyledons). This class therefore serves as the root of the taxonomic tree below.
